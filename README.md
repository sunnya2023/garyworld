## git add

- 사용법 : git add <추가할경로>

## git commit -m

- 사용법 : git commit -m "<메시지:따옴표 필수 아님>"
  `"적다가` 이 뒤에 엔터친 다음에 다시 치고 `"` 이거하면 여러줄로 입력됨

```
#git commit -m "여러 줄
>>
>> 첫번째 줄
>> 두번째 줄
>> 세번째 줄
>> "# git status
```

- 내 git의 현재상태를 본다

## git diff

- 이전 기록(commit)을 기준으로 변경사항을 본다
